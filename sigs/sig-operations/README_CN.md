# SIG operation

## SIG 简介

专注于提升OceanBase数据库的安装部署运维的用户体验，增强其对应产品OBD、obshell等工具功能丰富性、操作便利性与易用性，为数据库管理员、开发者提供强大、高效的运维与管理工具。
