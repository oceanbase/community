# 申请创建 oceanbase-skills 项目

## 项目基本信息

项目名称：oceanbase-skills

项目目的：oceanbase-skills 是一个面向 AI 编程助手（如 Cursor、Claude Code 等）的 Agent Skills 集合仓库，专为 OceanBase 开发者和用户设计。仓库以子目录形式组织多个独立的 Skill，每个 Skill 提供开箱即用的 SKILL.md 文件，可直接集成至 AI 编程工具中使用。

当前包含的 Skill：
- **oceanbase-deploy**：帮助 AI 助手引导用户完成 OceanBase 的部署与初始化配置，支持单机、分布式等多种部署模式。

后续规划：随着社区需求增长，将持续扩充更多 Skill，例如 SQL 调优、运维诊断、数据迁移、性能测试等，欢迎社区贡献者提交新的 Skill。面向的用户群体为使用 AI 编程工具进行 OceanBase 开发、部署、运维和集成的工程师。

项目责任人(github id)：

- Maintainer: gaoyunshan

开源协议：Apache 2.0

> 建议采用 Apache 2.0，如果不采用 Apache 2.0 协议，请注明原因

## 开源项目检查清单

> 检查清单是为了让项目更规范，让社区用户更容易使用。应该在项目创建后，尽快补充清单中的信息。

- [ ] 包含 README.md
- [ ] 工程类项目需要包含 CONTRIBUTING.md。参考 OceanBase 社区 [CONTRIBUTING 文件](https://github.com/oceanbase/.github/blob/main/CONTRIBUTING.md)
- [ ] 包含文件 CODE_OF_CONDUCT.md （没有此文件将使用社区现有的[行为准则文件](https://github.com/oceanbase/.github/blob/main/CODE_OF_CONDUCT.md) ）
- [ ] 工程类项目包含用户安装指导说明（通常在README.md中说明）
- [ ] 工程类项目包含用户使用指导说明（通常在README.md中说明）
- [ ] 工程类项目代码类项目包含编译指导说明（通常在README.md中说明）
