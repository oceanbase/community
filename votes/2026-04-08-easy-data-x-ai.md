# 申请 Fork Easy Data X AI 项目到 ob-labs

## 项目基本信息

原始项目地址：https://github.com/liboyang0730/easy-data-x-ai

项目用途：
Easy Data X AI 是面向所有 AI 爱好者的 Data 与 AI 基础知识入门教程。课程分为"道篇"和"术篇"两条学习路径：

- **道篇（悟其道）**：面向零基础 AI 爱好者和产品决策者，培养场景判断力、归因决策力和系统设计力
- **术篇（用其术）**：面向已能调用 LLM API 的开发者，提供坚实的工程基础、完整的数据层构建经验和 Agent 开发实战

课程涵盖 RAG 产品设计、记忆系统设计、Skill 与知识管理、Agentic RAG 实战、MCP 标准化等前沿主题，帮助开发者从理论到实践全面掌握 AI Agent 开发能力。

Fork 维护的原因：
将项目 Fork 到 ob-labs 组织下进行长期维护，便于：
1. 借助 OceanBase 社区的影响力，让更多开发者了解和学习 AI 与数据库结合的最佳实践
2. 与 OceanBase 生态更好结合，展示 AI Native 数据库在 Agent 场景中的应用
3. 获得社区支持，持续完善课程内容
