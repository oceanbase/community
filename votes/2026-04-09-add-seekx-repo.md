# 申请添加 seekx 代码仓库

## 项目基本信息

项目名称：seekx

项目目的：

seekx 是一款面向 **AI 智能体与人类用户** 的 **本地上下文检索引擎**：以用户文件为真实来源，通过索引实现「语义 + 关键词」的混合检索（BM25、向量检索、RRF 融合等），帮助开发者在文档、笔记、代码说明等场景中 **按含义查找内容**。

在 OceanBase 开源生态中的价值包括：

- 与 **SIG AI**「数据库与 AI 双向赋能」方向一致，可作为社区在 RAG、开发者工具链与知识库检索方向的补充。
- 工程形态清晰（TypeScript/Bun，OpenAI 兼容 Embedding API，本地 SQLite 索引），便于试用与集成。
- 支持增量索引与文件监听等能力；路线图含 MCP 等，可进一步服务 AI 工具生态。
