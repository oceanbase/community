# 申请创建 seekdb-client 项目

## 项目基本信息

项目名称：seekdb-client

项目目的：为 Cursor 等编辑器 提供 SeekDB 的插件支持，方便开发者在使用 Cursor 时便捷使用 SeekDB

项目责任人(github id)：

- Maintainer: Alfred-Lau

开源协议：Apache 2.0

> 建议采用 Apache 2.0，如果不采用 Apache 2.0 协议，请注明原因
